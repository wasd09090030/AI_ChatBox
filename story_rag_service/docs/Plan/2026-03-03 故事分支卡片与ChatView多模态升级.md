## 技术决策

| 项目 | 决策 |
|------|------|
| 图片 API 传输格式 | `data:{mimeType};base64,{base64}` 内嵌在 content array |
| 图片持久化 | 后端 `data/FileUpload/`，通过 `/files/{filename}` 静态访问 |
| URL 可达性检查 | 前端 `fetch(url, {method:'HEAD', mode:'no-cors'})` + 后端下载验证双重检查 |
| 多模态模型切换 | 有图片时自动建议使用 `deepseek-vl-chat`；无图片用配置默认模型 |
| localStorage 策略 | 会话消息只存 `{url}` 引用；base64 仅在内存中保留至发送完毕 |
| 用户认证 | `X-User-ID` 改为 Optional，后端直接读取 `settings.deepseek_api_key` |
